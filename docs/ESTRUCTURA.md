# Estructura Detallada

## `modules/aplicaciones`

Contiene los modulos mas completos o integradores.

- `auth-dashboard-frontend`: interfaz frontend con vistas de autenticacion y dashboard.
- `gestor-tareas-php-api`: aplicacion PHP con API, base de datos y gestion de tareas/comentarios.

## `modules/mini-proyectos`

Contiene proyectos pequeños nombrados por su funcionalidad:

- `formulario-registro`
- `landing-cardiologia`
- `utilidades-javascript`
- `gestor-tareas-colaborativo`
- `estado-cuenta-php`

## `modules/laboratorios-web`

Laboratorios y componentes web nombrados por lo que implementan:

- `fundamentos-html-css`
- `login-visual`
- `catalogo-peliculas-php`
- `registro-cursos-php`

## `database`

Scripts SQL centralizados para facilitar la instalacion.

## `archives`

ZIPs originales que venian dentro del repositorio. Se conservan como respaldo, pero no son necesarios para navegar los modulos reorganizados.
